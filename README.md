## simple boilerplate
#### Supports
  * SCSS
  * ES6 (TODO)
  * Gulp-uglify
  * Local JSON data

#### File tree
```
├── README.md
├── gulpfile.js
├── package.json
└── src
    ├── data
    │   └── dummy.json
    ├── images
    │   └── octocat.png
    ├── index.html
    ├── js
    │   └── master.js
    └── scss
        ├── master.scss
        └── partials
            └── _heading.scss
```

#### How to use
   run `npm install` to install modules

   run `npm start` to run gulpfile.js

   __Don't edit files in the public folder!__
